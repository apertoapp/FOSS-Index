# Multimedia

> Free and open-source software for image editing, video editing, audio production, and media playback.

| Icon | Operating System |
| --- | --- |
| 🪟 | Windows |
| 🍎 | macOS |
| 🐧 | Linux |

## Image Editing
- **GIMP**: Alternative to Photoshop for image processing 🪟 🍎 🐧 → [gimp.org](https://www.gimp.org/)
- **Krita**: Digital painting and illustration software 🪟 🍎 🐧 → [krita.org](https://krita.org/)
- **Inkscape**: Vector graphics editor (alternative to Illustrator) 🪟 🍎 🐧 → [inkscape.org](https://inkscape.org/)

## Video Editing
- **Kdenlive**: Non-linear video editor 🪟 🍎 🐧 → [kdenlive.org](https://kdenlive.org/)
- **Shotcut**: Intuitive and cross-platform video editor 🪟 🍎 🐧 → [shotcut.org](https://shotcut.org/)
- **Blender**: 3D modeling, animation, and video editing 🪟 🍎 🐧 → [blender.org](https://www.blender.org/)

## Audio
- **Audacity**: Audio editing and recording 🪟 🍎 🐧 → [audacityteam.org](https://www.audacityteam.org/)
- **Ardour**: Digital audio workstation (DAW) for music production 🪟 🍎 🐧 → [ardour.org](https://ardour.org/)

## Media Players
- **VLC**: Universal media player 🪟 🍎 🐧 → [videolan.org](https://www.videolan.org/vlc/)
- **MPV**: Minimalist and powerful media player 🪟 🍎 🐧 → [mpv.io](https://mpv.io/)
