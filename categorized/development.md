## Code Editors
- **VS Code (VSCodium)**: Modern code editor with extensions 🪟 🍎 🐧 → [vscodium.com](https://vscodium.com/)
- **Neovim**: Highly customizable text editor 🪟 🍎 🐧 → [neovim.io](https://neovim.io/)
- **Emacs**: Extensible editor (Lisp) for code, text, and more 🪟 🍎 🐧 → [gnu.org/software/emacs](https://www.gnu.org/software/emacs/)

## IDEs
- **Eclipse**: IDE for Java, C/C++, Python, and more 🪟 🍎 🐧 → [eclipse.org](https://www.eclipse.org/)
- **NetBeans**: Versatile IDE (Java, PHP, HTML, etc.) 🪟 🍎 🐧 → [netbeans.apache.org](https://netbeans.apache.org/)

## Version Control
- **Git**: Distributed version control system 🪟 🍎 🐧 → [git-scm.com](https://git-scm.com/)
- **GitLab**: Self-hosted Git platform 🪟 🍎 🐧 → [gitlab.com](https://gitlab.com/)

## Databases
- **PostgreSQL**: Advanced relational database 🪟 🍎 🐧 → [postgresql.org](https://www.postgresql.org/)
- **MySQL**: Popular relational database 🪟 🍎 🐧 → [mysql.com](https://www.mysql.com/)

## Containers and DevOps
- **Docker**: Containerization platform 🪟 🍎 🐧 → [docker.com](https://www.docker.com/)
- **Kubernetes**: Container orchestration 🪟 🍎 🐧 → [kubernetes.io](https://kubernetes.io/)
