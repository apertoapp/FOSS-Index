# Servers and Infrastructure

> Free and open-source software for web servers, monitoring, and collaboration tools.

## Web Servers
- **Apache HTTP Server**: Historic web server 🪟 🍎 🐧 → [apache.org](https://httpd.apache.org/)
- **Nginx**: High-performance web server and reverse proxy 🪟 🍎 🐧 → [nginx.org](https://nginx.org/)

## Monitoring
- **Prometheus**: Monitoring and alerting toolkit 🪟 🍎 🐧 → [prometheus.io](https://prometheus.io/)
- **Grafana**: Data visualization and dashboards 🪟 🍎 🐧 → [grafana.com](https://grafana.com/)

## Collaboration
- **Mattermost**: Open-source alternative to Slack 🪟 🍎 🐧 → [mattermost.com](https://mattermost.com/)
- **Nextcloud Talk**: Integrated video conferencing 🪟 🍎 🐧 → [nextcloud.com/talk](https://nextcloud.com/talk/)
