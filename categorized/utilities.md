## Web Browsers
- **Firefox**: Privacy-focused web browser 🪟 🍎 🐧 → [mozilla.org/firefox](https://www.mozilla.org/fr/firefox/)
- **LibreWolf**: Hardened Firefox for privacy 🪟 🍎 🐧 → [librewolf.net](https://librewolf.net/)

## Messaging
- **Thunderbird**: Email client by Mozilla 🪟 🍎 🐧 → [thunderbird.net](https://www.thunderbird.net/)
- **Signal**: End-to-end encrypted messaging 🪟 🍎 🐧 → [signal.org](https://signal.org/)

## File Management
- **Double Commander**: Dual-pane file manager 🪟 🍎 🐧 → [doublecmd.sourceforge.io](https://doublecmd.sourceforge.io/)
