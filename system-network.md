# System and Network

> Free and open-source operating systems, virtualization tools, network utilities, and security software.

## Operating Systems
- **Ubuntu**: Stable Linux distribution 🐧 → [ubuntu.com](https://ubuntu.com/)
- **Debian**: Reliable Linux distribution 🐧 → [debian.org](https://www.debian.org/)
- **Tails**: Privacy-focused OS (live USB) 🐧 → [tails.boum.org](https://tails.boum.org/)

## Virtualization
- **QEMU**: Emulation and hardware virtualization 🪟 🍎 🐧 → [qemu.org](https://www.qemu.org/)
- **VirtualBox**: Virtualization for workstations 🪟 🍎 🐧 → [virtualbox.org](https://www.virtualbox.org/)

## Network
- **Wireshark**: Network protocol analyzer 🪟 🍎 🐧 → [wireshark.org](https://www.wireshark.org/)
- **Nextcloud**: Self-hosted cloud (files, contacts, calendar) 🪟 🍎 🐧 → [nextcloud.com](https://nextcloud.com/)

## Security
- **KeePassXC**: Offline password manager 🪟 🍎 🐧 → [keepassxc.org](https://keepassxc.org/)
- **WireGuard**: Modern and fast VPN solution 🪟 🍎 🐧 → [wireguard.com](https://www.wireguard.com/)
