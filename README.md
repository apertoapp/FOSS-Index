- **Taiga** : Outil agile (Scrum/Kanban) pour les équipes 🪟 🍎 🐧 → [taiga.io](https://taiga.io/)
- **OpenProject** : Gestion de projets avec suivi des tâches, Gantt, et collaboration 🪟 🍎 🐧 → [openproject.org](https://www.openproject.org/)
- **Redmine** : Gestion de projets flexible avec suivi des bugs 🪟 🍎 🐧 → [redmine.org](https://www.redmine.org/)
